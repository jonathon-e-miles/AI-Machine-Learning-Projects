Census Income Classification & Demographic Analysis
Project Overview

This project focuses on the processing and classification of US Census Bureau data to predict income levels based on demographic variables. It demonstrates a high-precision approach to handling large, multi-featured datasets where accuracy and data integrity are paramount.
Key Data Analyst Workflows

    ETL & Data Cleaning: Utilized Pandas to handle missing values, encode categorical variables (such as education and occupation), and normalize numerical features.

    Demographic Feature Engineering: Analyzed the relationship between age, marital status, and capital gain/loss to identify the most significant predictors of socioeconomic status.

    Statistical Analysis: Performed exploratory data analysis (EDA) to ensure the dataset was balanced and representative before applying classification models.

Technical Stack

    Languages: Python (Pandas, NumPy, Scikit-Learn)

    Environment: Jupyter Notebooks

    Core Concepts: Data Ingestion | Feature Scaling | Classification Evaluation
