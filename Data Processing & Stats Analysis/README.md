Project Overview

This repository contains a suite of Python-based workflows dedicated to the Extraction, Transformation, and Loading (ETL) of complex tabular datasets. The primary focus of these notebooks is to ensure data integrity through rigorous cleaning, normalization, and exploratory statistical analysis.
Core Data Analyst Competencies

* Pipeline Engineering: Developed modular Python scripts to automate the ingestion of raw data from multiple sources, ensuring consistent formatting and structural alignment.

* Advanced Data Cleaning: Utilized Pandas and NumPy to resolve null values, handle outliers, and perform vectorised operations on datasets with 10,000+ rows.

* Statistical Validation: Performed descriptive and inferential statistical testing to identify correlations, variance, and data distributions before downstream modeling.

* Documentation & Reproducibility: Leveraged Jupyter Notebooks to provide a transparent, step-by-step audit trail of all data transformations—essential for high-stakes, nonpartisan reporting.

Technical Stack

* Languages: Python (Pandas, NumPy, Scipy.stats)

* Visualization: Matplotlib | Seaborn

* Environment: Jupyter Notebooks | Git Version Control

* Data Visualization: Utilized Matplotlib and Seaborn to create correlation heatmaps and demographic distribution plots, ensuring the data's "story" is clear and statistically sound.
